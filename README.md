# Горячие клавиши
#### Копировать: 
CtrL+C
#### Вставить: 
CtrL+V
#### Сохранить: 
CtrL+S
#### Вырезать: 
CtrL+X
#### Отмена: 
Ctrl+Z

# -git clone <url>
### Скачать проект:
```bash
git add. <URL>
```
### Учесть все файлы:
```bash
git add .
```
### Сохранить коммит:
```bash
git commit -m"..."
```
### Отправка в облаке:
```bash
git push
```
